## OrangeHRM Automation Testing Project
#📌 Project Description

The OrangeHRM Automation Testing Project is designed to test the OrangeHRM demo web application using Selenium WebDriver.
It ensures that core HR functionalities like login, dashboard access, and logout are working correctly.

This project focuses on validating UI functionality, input fields, and system behavior under different test conditions.

#🎯 Objective
To automate and validate OrangeHRM login functionality
To ensure dashboard loads correctly after successful login
To verify logout functionality
To identify bugs and improve application quality

#🧩 Modules Covered
Login Module
Dashboard Module
Logout Module
User Interface Validation

#🧪 Testing Scope
Functional Testing
UI Testing
Positive Testing
Negative Testing
Validation Testing
